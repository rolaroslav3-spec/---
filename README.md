Это авторская экосистема, написанная с нуля. В основе — математическая модель времени, температуры и биологических циклов.

## Как следить за прогрессом
Я веду разработку итеративно: каждый день добавляю по одной новой логике. История всех изменений доступна в разделе Commits.
